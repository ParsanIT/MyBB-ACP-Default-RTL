## Intro

MyBB 1.8 ACP(Admin Control Panel) Theme in RTL version for RTL languages. (based on Default Theme that exist in MyBB 1.8.39 Package)  
This RTL version is fully compatible and is the best in terms of details. ;)

### How to use?

1. Extract current release package .zip file to your mybb root. (or Upload files with current structure to your mybb root.)
2. Go to ACP(Admin Control Panel) -> Configuration (tab) -> Settings (sidebar) -> Control Panel Preferences (Global)
3. In "Control Panel Style" select "parsanit_default_rtl"

For see changes may need to refresh the page.

#### If it doesn't changed to RTL version... you should change your personal prefrences

4. Go to ACP(Admin Control Panel) -> Home (tab) -> Preferences (sidebar)
5. In "Admin Control Panel Theme" select "Use Default" or "parsanit_default_rtl"

For see changes may need to refresh the page.
